Welcome to the Women Writers Vector Toolkit (WWVT). This site can be considered a text analysis "laboratory" for experimenting with word embedding models. The word vector analysis tools on this site use texts from the Women Writers Online corpus, making the space particularly useful for academics researching early modern literature, gender, and history;, though all are welcome to use the toolkit.

We aim to make this site an accessible and transparent space for users with assorted expertises. We have supplied this site with rich documentation in order to: share the process and choices behind the regularization of the texts, publish the coding languages used to create particular models for others to use, and write up assignments and case studies based on our analyses to provide visitors with examples and results from our own research.Read more about thebackground of the project, how to navigate this site, and the project team.
