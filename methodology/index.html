Downloads & Code

The Tool

Code for the model querying tool can be found on GitHub at https://github.com/NEU-DSG/wwp-w2vonline. The tool itself is an RStudio Shiny application developed by Jonathan D. Fitzgerald and Parth Tandel. It uses Benjamin Schmidt's R package wordVectors to create and query models of words.

Corpora

The Women Writers Online (WWO) corpus is freely available on request. For consideration, send an email with a brief description of your research to wwp@northeastern.edu.

The Victoria Women Writers Project (VWWP) corpus

The Early English Books Online Text Creation Partnership (EEBO-TCP) corpus

Each corpus was transcribed and encoded in XML, following the Text Encoding Initiative (TEI) guidelines. These XML files were then transformed into normalized plain text using XSLT and XQuery scripts.
