Methodology

The Tool

Code for the model querying tool can be found on GitHub at https://github.com/NEU-DSG/wwp-w2vonline. The tool itself is an RStudio Shiny application developed by Jonathan D. Fitzgerald and Parth Tandel. It uses Benjamin Schmidtâ€™s R package wordVectors to create and query models of words.

The GitHub repository includes copies of the models, as well as a JSON catalog of those models and instructions for running the application yourself.

Corpora

The Women Writers Online (WWO) corpus represents works published between 1400 and 1850 which were authored, translated, or compiled by women. The corpus is available online to subscribing institutions or individuals. The XML files are freely available on request. For consideration, send an email with a brief description of your research to wwp@neu.edu.

The tool also includes documents from the Victoria Women Writers Project (VWWP) corpus and the Early English Books Online Text Creation Partnership (EEBO-TCP) corpus.

Each corpus was transcribed and encoded in XML, following the Text Encoding Initiative (TEI) guidelines. For this project, the XML files were then transformed into normalized plain text using XSLT and XQuery scripts.
