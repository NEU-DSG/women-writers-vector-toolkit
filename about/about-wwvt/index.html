About the Women Writers Vector Toolkit

This project is a result of the Northeastern Tier 1 Seedling Grant titled "Word Vector Analysis for TEI/XML: A user-friendly toolkit" and is part of the Women Writers Project. This project merges expertises across the Humanities, Information Sciences, and Computer Science disciplines, from the graduate to the professional level, to create a pedagogical space that documents the process of analyzing TEI (Text Encoding Initiative) and XML (eXtensible Markup Language) encoded corpora using word vector models. This toolkit demonstrates how to transform TEI/XML encoded data using XSLT and XQuery, create word vector models based on those results, and implement different pedagogical approaches in the classroom using these word vector models. While this project uses the Women Writers Online corpus, the process for reshaping the TEI/XML encoded documents and for creating word vector models can be applied to different TEI/XML encoded corpora. We hope to be a platform and example for others who wish to implement similar tools for their own corpora.



Since 1988, The Women Writers Project has been a project devoted to early modern women's writing and electronic text encoding. Their goal is to bring texts by pre-Victorian women writers out of the archive and make them accessible to a wide audience of teachers, students, scholars, and the general reader. They support research on women's writing, text encoding, and the role of electronic texts in teaching and scholarship.
